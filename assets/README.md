# Assets Directory

This directory contains application assets.

## Required Files

### icon.ico

Application icon file for Windows executable.

**Requirements:**

- Format: ICO (Windows Icon)
- Recommended sizes: 16x16, 32x32, 48x48, 64x64, 128x128, 256x256 (multi-resolution)
- You can create one using online tools like:
  - https://icoconvert.com/
  - https://convertio.co/png-ico/

**Note:** If no icon is provided, the build will use the default PyInstaller icon.
